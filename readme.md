# Playwright Test Suite

Follow these steps to set up your environment and run the automated tests for
<a href="https://filip-peev.com/your-notes/" target="_blank">Your Notes App</a>.

## Prerequisites

You will need the following installed on your system - Node.js and Playwright

---

## Installation Guide

### 1. Create Project Directory
Create a folder on your local machine where the tests will reside
```
c:\my_tests
```
### 2. Install Node.js
1. Go to the <a href="https://nodejs.org/en" target="_blank">Node.js Website</a>.
2. Download and run the installer for your operating system.
3. Follow the installation prompts until finished.

### 3. Initialize Playwright
1. Open your Command Prompt (`cmd`).
2. Navigate to your folder
```
c:\my_tests
```
3. Once inside it, run this command to install Playwright:<br>
```
npm init playwright@latest
```
### 4. Get the tests archive
1. Download the tests archive from <a href="https://github.com/Filip-Peev/Your_Notes_Tests/releases/download/v1.0/_your-notes.zip" target="_blank">Automation Tests</a>.
2. Extract it somewhere and take the folder and file
```
tests
playwright.config.ts
```
and put them inside, replacing files and folders
```
c:\my_tests
```
